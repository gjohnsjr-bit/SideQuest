# SideQuest

SideQuest is a gamified adulting app designed to help users navigate the complexities of adulthood in a fun and engaging way. 

## Features
- **Task Management**: Create and manage tasks that reflect important aspects of adult life.
- **Rewards System**: Earn points and rewards for completing tasks and milestones.
- **Community Challenges**: Participate in community challenges to stay motivated and accountable.

## Installation
To get started with SideQuest, clone the repository and install the required dependencies:
```bash
git clone https://github.com/gjohnsjr-bit/SideQuest.git
cd SideQuest
npm install
```

## Usage
After installing the dependencies, you can run the app locally:
```bash
npm start
```
